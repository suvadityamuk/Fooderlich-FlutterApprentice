# Fooderlich - Flutter Apprentice
An end-to-end Flutter app built to get details and trending recipes - made as a part of the book 'Flutter Apprentice' by raywenderlich.com

This application was made while following across the 20 chapters of Flutter Apprentice, a book by raywenderlich.com

To use, you must use Android Studio to compile, build and run it on an emulator/physical device.
This app also uses the [Edamam API](https://developer.edamam.com/edamam-recipe-api). Use this to generate API Keys and run the application.
